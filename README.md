Project Overview  

This project presents the design of a 14-storey commercial building developed using AutoCAD.  
The objective of the project was to apply computer graphics principles in architectural drafting and building visualization.

The design demonstrates practical understanding of:

- 2D drafting techniques  
- Architectural floor planning  
- Structural layout organization  
- Commercial space planning  
- Use of layers and scaling in AutoCAD  

Project Description  

The building design includes:

- Multiple commercial office floors  
- Structural layout planning  
- Organized spatial distribution  
- Professional drafting standards  

The project was developed to reflect real-world commercial architectural standards.

Software Used  

- AutoCAD  

Project Files  

This repository contains:

- AutoCAD drawing files (.dwg)  
- Project documentation  

Objective of the CAT  

To demonstrate competency in computer graphics through the design of a multi-storey commercial building using AutoCAD tools and techniques.

Submission  

This project is submitted as part of the Computer Graphics Continuous Assessment Test (CAT) requirements.
